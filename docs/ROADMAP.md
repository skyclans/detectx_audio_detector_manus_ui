### 2.2 RTO 보조 엔진 통합
**우선순위: 중간**

| 작업 | 효과 |
|------|------|
| CNN 후 RTO 검증 추가 | FP 70% 추가 복구 |
| RTO < 0.10 → Human 확정 | AND 로직으로 엄격화 |

**예상 파이프라인:**
```
CNN >= 0.9 → AI 후보 → RTO 검증
  ├─ RTO >= 0.10 → AI 확정
  └─ RTO < 0.10 → Human 복구
```
